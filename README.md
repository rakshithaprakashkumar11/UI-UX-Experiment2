# UI-UX-Experiment2

## Aim:
Designing a Responsive Mobile and Web Experience for ShopSmart:
ShopSmart is a modern e-commerce platform focused on smart budgeting and seamless online shopping. The goal of this design case study is to create intuitive mobile and web layouts using strong UI/UX principles, effective interaction behavior, and a consistent brand identity featuring the ShopSmart logo.

## Algorithm:
## a. Applying Gestalt Principles in ShopSmart Layout Design
To create a visually organized and user-friendly interface, the following Gestalt principles are applied:

🔹 1. Proximity
Products are structured into clear product cards where image, product name, price, rating, and “Add to Cart” button are grouped together. This helps users quickly scan and understand product information.

🔹 2. Similarity
All call-to-action buttons use the same purple theme with rounded corners. Icons (cart, search, profile) maintain consistent line style and thickness across screens.

🔹 3. Figure–Ground
The interface uses a soft lavender background while primary elements (ShopSmart logo, product cards, and CTAs) stand out in purple and yellow accents, ensuring clear visual focus.

🔹 4. Continuity
Mobile layout follows a smooth vertical scroll structure, while the web layout follows a structured grid alignment. This ensures natural eye movement and seamless browsing.

🔹 5. Visual Hierarchy
Large bold fonts for offers and discounts
Medium-sized fonts for product titles
Smaller, lighter fonts for descriptions and secondary details
This helps users prioritize information efficiently.

## c. Interaction Behavior & Usability Principles

✅ Mobile-First Approach
Design starts with mobile and scales to larger screens using responsive techniques.

✅ Real-Time Feedback
Toast notification: “Item added to cart successfully”

Animated cart icon when product is added

✅ Error Prevention
Confirmation before removing items

Undo option after cart deletion

✅ Consistency
Same color scheme, icon style, and button placement across all pages.

✅ Accessibility
High contrast color ratio
Readable typography
Large touch-friendly buttons
Screen-reader friendly labels

✅ User Control & Freedom
Users can:
Modify cart quantity easily
Save items for later
Track orders in real-time
